# EIGRP Routing Lab

## Objective
Implement dyanmic routing using EIGRP with authentication.

## Features
- Key chain
- Automatic route advertisement
- Neighbor discovery
- Route learning

## Verification
  
  show ip eigrp neighbors
  
  show ip route

  Succesful end-to-end connectivity.
